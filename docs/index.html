Welcome to ModelTrace

ModelTrace is a research-oriented framework for analyzing, debugging, and evaluating machine learning models.

Use the links above to explore:

  • Documentation: Overview, working process, future work, and academic relevance.
  • Experiments: All eight experiments with methodology.
  • API Reference: ModelInspector class and methods.
  • README: Installation, usage, folder structure, and example outputs.

Quick Start

git clone https://github.com/MANISH-K-07/ModelTrace.git
cd ModelTrace
pip install -r requirements.txt
python setup.py install

# Run all experiments
python examples/run_all_experiments.py
